css类元素 .开头 html中引用 要元素对应相同的class

如 css中 .xxx 的样式  html中引用要为 <a class = "xxx">



| css选择器类型 | 语法    | HTML对应属性   | 示例             |
| :------------ | :------ | :------------- | :--------------- |
| ID选择器      | `#name` | `id="name"`    | `#sender-column` |
| 类选择器      | `.name` | `class="name"` | `.sender-column` |
